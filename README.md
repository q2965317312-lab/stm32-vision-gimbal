# STM32 Vision Gimbal

基于 STM32F407、TMC2209 和 MaixCAM2 的二轴视觉跟踪云台项目。

## Features

- STM32F407 HAL 工程
- TMC2209 双轴步进电机驱动
- MaixCAM2 视觉识别与 UART 数据输出
- OLED 状态显示
- 按键调试
- PID 目标跟踪
- 硬件调试与故障排查记录

## Status

已完成硬件复刻、供电调试、OLED/按键验证、双轴步进电机测试和基础跟踪功能调试。
