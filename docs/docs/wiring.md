# Wiring

## Power

| Module | Voltage | Note |
| --- | --- | --- |
| STM32F407 board | 5V | Buck output adjusted to about 4.97V |
| STM32 IO | 3.3V | Generated by onboard regulator |
| TMC2209 VM | 12V | Motor power |
| MaixCAM2 | 5V | Powered separately or by Type-C |

## Stepper Driver Pins

| Function | STM32 Pin | Description |
| --- | --- | --- |
| Yaw STEP | PA6 | Step pulse |
| Yaw DIR | PC4 | Direction |
| Yaw EN | PA7 | Enable, active low |
| Pitch STEP | PE9 | Step pulse |
| Pitch DIR | PE10 | Direction |
| Pitch EN | PE8 | Enable, active low |

## MaixCAM2 UART

| MaixCAM2 | STM32F407 |
| --- | --- |
| TX | UART5_RX / PD2 |
| RX | UART5_TX / PC12 |
| GND | GND |

Baud rate: 115200
